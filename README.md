# Interview Simulator

A modern interview preparation platform built using React, TypeScript, Tailwind CSS, and Netlify.

## Features

- Role-based interview simulations
- Interactive interview sessions
- Session scoring and feedback
- Responsive UI
- Smooth animations and modern design

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS
- TanStack Router
- Vite
- Netlify
