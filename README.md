# 🎤 Interview Simulator

A modern and interactive interview preparation platform built using **React, TypeScript, Tailwind CSS, and Netlify**.

🚀 Practice interviews, improve communication skills, build confidence, and track your performance through realistic interview simulations.

---

## 🌐 Live Demo

https://interview-simulatorr.netlify.app/

## ✨ Features

✅ Role-Based Interview Simulations

✅ Interactive Interview Sessions

✅ Session Scoring & Feedback

✅ Real-Time Question Flow

## 🔄 Project Workflow

### 1️⃣ Select Interview Preferences

The user selects:
- Job Role
- Interview Style
- Interview Session

⬇️

### 2️⃣ Start Interview

The simulator launches an interactive interview session with realistic questions.

⬇️

### 3️⃣ Answer Questions

Users respond to interview questions one by one in a structured flow.

⬇️

### 4️⃣ Performance Evaluation

The system analyzes the interview session and generates performance insights.

⬇️

### 5️⃣ Session Score

Users receive:
- 📊 Session Score
- 💡 Feedback
- 📈 Performance Summary
